Design at Ui

Desain User Interface (UI) adalah proses yang digunakan desainer untuk membuat tampilan
dalam perangkat lunak atau perangkat terkomputerisasi, dengan fokus pada tampilan atau
gaya. Tujuan dari desainer UI adalah untuk membuat desain antarmuka yang membuat
pengguna mudah untuk digunakan dan menyenangkan.
